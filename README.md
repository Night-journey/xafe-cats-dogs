# 西安财经大学流浪动物关爱平台（xafe-cats-dogs）

一个面向校园流浪猫狗关爱与协作的静态 Web 项目，提供 **地图标注、猫狗图鉴、论坛交流、委托协助、领养信息、救助知识** 等功能模块，帮助同学们更高效地参与校园动物救助。

## 项目亮点

- 🐾 **多页面协作平台**：首页、地图、图鉴、论坛、委托、领养、救助知识、关于我们等页面。
- 🗺️ **校园地图可视化**：基于 Leaflet + 高德瓦片展示校园内猫狗/喂食点/安置点。
- 💾 **零后端可运行**：主要数据保存在浏览器 `localStorage`，开箱即用，适合课程作业与原型演示。
- 📱 **移动端适配**：包含移动导航与响应式布局。
- 🔐 **管理入口**：提供管理员登录与后台页面（当前为前端演示方案）。

## 技术栈

- HTML5
- CSS3
- 原生 JavaScript（Vanilla JS）
- Leaflet（地图）
- Vercel（部署配置）

## 快速开始

### 方式一：直接双击打开（最简单）

直接在浏览器中打开 `index.html` 即可浏览大部分页面。

### 方式二：本地静态服务器（推荐）

> 建议使用本地服务器，避免部分浏览器对本地文件策略造成的功能限制。

使用 Python：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

## 功能说明

- **首页**：展示平台定位、快捷入口、统计信息与动态。
- **校园地图**：展示不同类型点位（猫、狗、喂食点、安置点）并支持筛选。
- **猫狗图鉴**：查看校园动物档案信息。
- **论坛**：同学交流经验、提问与分享。
- **委托**：发布喂食、送医、募捐等求助任务。
- **领养**：发布与浏览待领养动物信息。
- **救助知识**：提供基础救助指南与科普内容。
- **管理员后台**：用于演示信息管理流程。

## 数据存储说明

当前项目使用浏览器本地存储：

- 读取键：`xafeData`
- 管理员登录状态键：`isAdmin`

清理浏览器缓存或 `localStorage` 后，数据会恢复为默认初始化数据。

## 部署

项目包含 `vercel.json`，可直接部署到 Vercel。

一般流程：

1. 推送仓库到 GitHub。
2. 在 Vercel 导入仓库。
3. 按默认静态站点配置部署。
