# 🐱 西安财经大学流浪动物关爱平台

用爱守护每一个生命，让校园充满温暖。

## 🌐 在线预览

访问 GitHub Pages: https://night-journey.github.io/xafe-cats-dogs/

## 📱 功能特点

### 🗺️ 校园地图
- 查看校园内流浪猫狗的出没地点
- 标记喂食点和休息区
- 提交新的地点信息

### 💬 论坛
- 分享救助经验
- 晒照交流
- 问答互动

### 📋 委托
- 发布喂食委托
- 医疗帮助请求
- 捕捉送医协助
- 临时安置需求

### 🏠 领养
- 发布领养信息
- 浏览待领养动物
- 联系领养

### 📚 救助知识
- 救助技巧
- 喂养指南
- 安全注意事项

## 🚀 技术栈

- 纯静态 HTML + CSS + JavaScript
- 无需后端，数据存储在浏览器本地存储
- GitHub Pages 免费托管

## 📂 项目结构

```
xafe-cats-dogs/
├── index.html      # 首页
├── map.html        # 校园地图
├── forum.html      # 论坛
├── entrust.html    # 委托
├── adoption.html   # 领养
├── rescue.html     # 救助知识
├── about.html      # 关于我们
├── css/
│   └── style.css   # 样式文件
└── js/
    ├── main.js     # 主要脚本
    └── data.js     # 数据文件
```
